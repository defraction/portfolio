Scratch Project

CSP August 30, 2018 - September 8, 2018

This project is a very simple platformer built from Scratch which is a drag and drop coding laguage which anyone can use. The objective of the project was to familarize ourselves and experiment with Scratch and code a game or story via Scratch. One important aspect of the project was to figure out how to make platformer physics using Scratch. This problem was solved by using key components of platformers from other projects on the website. My role in the project was mainly to do the coding while my partner did the documentation, but we both worked on each thing a bit.
